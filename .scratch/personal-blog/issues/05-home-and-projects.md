# Home and projects

## Parent

`.scratch/personal-blog/PRD.md`

## What to build

Build the personal homepage and projects page. The homepage should introduce the blog owner, surface featured posts, recent posts, featured projects, social links, and common topic tags. The projects page should render structured project content.

## Acceptance criteria

- [x] The homepage includes a personal hero with avatar, name or nickname, short introduction, and social links.
- [x] The homepage shows featured posts.
- [x] The homepage shows recent posts.
- [x] The homepage shows featured projects.
- [x] The homepage surfaces common topic tags.
- [x] `/projects` renders structured project entries.
- [x] Project cards show status, tags, and optional repository/demo links.
- [x] The homepage and projects page follow the shared Mizuki-inspired visual system.
