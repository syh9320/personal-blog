# Blog reading experience

## Parent

`.scratch/personal-blog/PRD.md`

## What to build

Build the core blog browsing and reading experience: full blog index, article pages, tag pages, archive page, article metadata display, table of contents behavior, and polished code block rendering.

## Acceptance criteria

- [x] `/blog` lists published posts in a readable, responsive layout.
- [x] Article detail pages render MDX content with strong typography.
- [x] Article pages show title, description, publication date, optional updated date, and tags.
- [x] `/tags` lists all available tags.
- [x] `/tags/[tag]` lists posts for a tag.
- [x] `/archive` groups posts chronologically.
- [x] Article pages support a table of contents that works well on desktop and mobile.
- [x] Code blocks have syntax highlighting that works in light and dark themes.
- [x] Code blocks support copy actions and polished overflow behavior.
