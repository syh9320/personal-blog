# Project scaffold

## Parent

`.scratch/personal-blog/PRD.md`

## What to build

Create the initial Astro project foundation for the personal blog. The site should run locally, build successfully, support MDX, use Tailwind CSS, and expose a central site configuration for personal information, navigation, SEO defaults, theme defaults, comments, and analytics.

## Acceptance criteria

- [x] The project has an Astro application scaffold suitable for a static personal blog.
- [x] MDX support is installed and available for content.
- [x] Tailwind CSS is installed and wired into the Astro project.
- [x] A central site configuration exists for identity, navigation, SEO defaults, theme, comments, and analytics.
- [x] The default language is configured as `zh-CN`.
- [x] Local development can start successfully.
- [x] A production build completes successfully.
