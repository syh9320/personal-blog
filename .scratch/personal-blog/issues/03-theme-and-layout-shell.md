# Theme and layout shell

## Parent

`.scratch/personal-blog/PRD.md`

## What to build

Build the Mizuki-inspired visual foundation and shared layout shell. The site should have refined card surfaces, soft colors, responsive spacing, a top navigation area, a lightweight desktop sidebar where appropriate, and complete light/dark/system theme behavior.

## Acceptance criteria

- [x] The visual system uses CSS variables and Tailwind tokens for colors, surfaces, borders, shadows, and text.
- [x] Light, dark, and system theme modes are supported.
- [x] Theme choice persists across sessions.
- [x] First paint avoids obvious theme flashing.
- [x] A shared page shell provides top navigation and responsive layout structure.
- [x] Desktop layouts can include a lightweight sidebar without harming the reading column.
- [x] Mobile layouts collapse into a clean single-column flow.
- [x] The design direction feels refined, card-based, and Mizuki-inspired without directly copying the Mizuki theme.
